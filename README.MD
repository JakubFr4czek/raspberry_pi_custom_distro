# RaspberryPi 4 Poky-based distribution build with embedded minecraft server

## How to run
1. Make sure you have docker installed
2. Clone and cd ino repository
3. run ./dockerize.sh
4. run ./build.sh
5. run ./extract_image.sh
6. Image will be copied to your working directory. 
7. Upload the image to you RaspberryPi.
